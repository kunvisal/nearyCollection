## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repo-url>
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view the app.

## Development Cycle

1.  **Create a Branch**: `git checkout -b feature/my-new-feature`
2.  **Code**: Implement your changes.
3.  **Test**: Verify locally.
4.  **Format**: `npm run lint` and `npm run format`.
5.  **Commit**: Use descriptive commit messages.
6.  **Push**: `git push origin feature/my-new-feature`.
7.  **PR**: Open a Pull Request for review.
